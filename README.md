Daily Expense Tracker

A simple, friendly tool to help you understand where your money goes—day by day, dollar by dollar.

What This Project Is About

Managing finances can feel overwhelming. This Daily Expense Tracker is designed to make it easier.
It gives you a clean, intuitive way to log your expenses, categorize them, and review your spending patterns without the stress of spreadsheets or complex budgeting apps.
Whether you’re trying to save money, build better habits, or simply stay aware of your daily spendings, this tracker supports you with clarity and simplicity.

Key Features
Add daily expenses with amount, category, note, and date

Organized categories like Food, Travel, Bills, Shopping, and more

Dashboard summary to visualize totals and trends

Why I Built This
Most of us have had that moment: “Where did all my money go?”
This project was born from exactly that feeling. I wanted something small, lightweight, and personal—a tool that doesn’t try to do too much, but does exactly what’s needed.
